# 🍔 Foodie

A modern food ordering and restaurant discovery web application designed to provide users with a smooth and interactive food browsing experience.

---

## 🌟 Features

- 🍕 Browse delicious food items
- 🛒 Add to cart functionality
- 🔍 Search and filter food items
- 📱 Responsive design for mobile & desktop
- ⚡ Fast and user-friendly interface
- 🍟 Restaurant and fast-food listing
- ❤️ Attractive UI/UX

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
